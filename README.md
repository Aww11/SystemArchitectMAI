# Домашнее задание 01: Интернет-магазин (Вариант 2)

## Выбранный вариант
**Вариант 2: Магазин** (по аналогии с https://ozon.ru/)

## Описание системы
Система представляет собой интернет-магазин, где пользователи могут:
- Просматривать товары
- Добавлять товары в корзину
- Оформлять заказы

## Роли пользователей
1. **Покупатель** — ищет товары, добавляет в корзину, оплачивает заказы
2. **Продавец** — добавляет новые товары, управляет ассортиментом

## Внешние системы
1. **Платежная система** — обработка платежей по картам
2. **Email сервис** — отправка уведомлений о заказах
3. **SMS сервис** — отправка кодов подтверждения

## Контейнеры системы
1. **Веб-приложение** (React.js) — клиентская часть для браузера
2. **Мобильное приложение** (React Native) — клиентская часть для телефонов
3. **API Gateway** (Kong/Nginx) — единая точка входа для всех запросов
4. **Сервис пользователей** (GoLang) — регистрация и управление пользователями
5. **Сервис товаров** (GoLang) — управление каталогом товаров
6. **Сервис корзины** (GoLang) — управление корзиной покупок
7. **Сервис заказов** (GoLang) — оформление заказов
8. **База данных пользователей** (PostgreSQL) — хранение данных пользователей
9. **База данных товаров** (PostgreSQL) — хранение каталога
10. **База данных корзин** (PostgreSQL) — хранение содержимого корзин
11. **База данных заказов** (PostgreSQL) — хранение заказов

## Диаграммы
- System Context (C1) — общий контекст системы
- Container (C2) — внутреннее устройство
- Dynamic — сценарий "Добавление товара в корзину"
